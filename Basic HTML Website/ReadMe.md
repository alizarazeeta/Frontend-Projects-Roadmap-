# Basic HTML Website

---

In this project, you are required to create a simple HTML only website with multiple pages. The website should have following pages:

---

- [x] Homepage

- [x] Projects

- [x] Articles

- [x] Contact

The website should have a navigation bar that should be present on all pages and link to all the pages.

You are not required to style the website, you are only required to create the structure of the website using HTML. Goals of this project are:

- [x] Learn how to create multiple pages in a website.

- [x] Structure a website using HTML in a semantic way.

- [x] Structure in a way that you can easily add styles later.

- [x] Add SEO meta tags to the website.

##Conclusion

After completing this project, you will have a good understanding of how to structure a website using HTML, basic SEO meta tags, HTML tags, forms etc. You can now move on to the next project where you will learn how to style this website using CSS.
